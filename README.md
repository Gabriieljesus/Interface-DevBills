# 💹 DevBills - Gestão Financeira Inteligente

O **DevBills** é uma aplicação Full Stack moderna desenvolvida para facilitar o controle financeiro pessoal. Com uma interface intuitiva e gráficos interativos, você pode gerenciar receitas, despesas e acompanhar a evolução do seu saldo de forma simples e eficiente.

---

🔗 **Acesse o projeto online:** [https://dev-bills-2026.vercel.app/](https://dev-bills-2026.vercel.app/)

---

## 🚀 Tecnologias e Ferramentas

O projeto utiliza o que há de melhor no ecossistema JavaScript/TypeScript para garantir performance e segurança.

### **Front-end**
| Tecnologia | Finalidade |
| :--- | :--- |
| **React + Vite** | Interface rápida e otimizada. |
| **TypeScript** | Tipagem estática para evitar erros e melhorar a manutenção. |
| **Tailwind CSS** | Estilização responsiva e moderna com classes utilitárias. |
| **Recharts** | Visualização de dados através de gráficos dinâmicos. |
| **Lucide Icons** | Conjunto de ícones leves e elegantes. |
| **Firebase Auth** | Autenticação segura com login via Google. |

### **Back-end**
| Tecnologia | Finalidade |
| :--- | :--- |
| **Fastify** | Framework web de alta performance para Node.js. |
| **Prisma ORM** | Comunicação segura e tipada com o banco de dados. |
| **MongoDB** | Banco de dados NoSQL escalável para transações. |
| **Zod** | Validação rigorosa de esquemas e dados da API. |
| **Biome** | Formatação e linting de código de última geração. |

---

## ✨ Funcionalidades

- 🔑 **Login Social:** Autenticação rápida e segura via Google através do Firebase.
- 📊 **Dashboard Visual:** Resumo financeiro com cards de Entradas, Saídas e Saldo Total.
- 📈 **Gráficos:** Análise de despesas por categoria e histórico mensal.
- 🔍 **Filtros Avançados:** Busca de transações por nome, mês e ano.
- 📝 **Gestão de Transações:** Criação e exclusão de receitas e despesas.
- 🛡️ **Privacidade:** Cada usuário visualiza apenas seus próprios dados.
- 🌑 **Design Moderno:** Interface responsiva com Dark Theme e ótima usabilidade.

## 🛠️ Instalação e Uso

### **1. Clone o repositório**
```bash
# Clone o Front-end
git clone [https://github.com/Gabriieljesus/Interface-DevBills.git](https://github.com/Gabriieljesus/Interface-DevBills.git)

# Clone o Back-end
git clone [https://github.com/Gabriieljesus/Api-DevBills.git](https://github.com/Gabriieljesus/Api-DevBills.git)

2. Configuração do Backend

cd Api-DevBills
npm install
npx prisma generate
npm run dev

3. Configuração do Frontend

cd ../Interface-DevBills
npm install
npm run dev
```
